== Description ==

The Finanzinform plugin includes a ticker from the most significant stock exchanges by [http://www.finanzinform.de/](http://www.finanzinform.de/).
The plugin is a high configurable tool for all financial blogs.

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==

1. Extract `finanzinform.zip`.
2. Upload the `finanzinform`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
